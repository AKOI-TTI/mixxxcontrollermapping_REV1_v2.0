# Mixxx Controller Mappings for REV1 v2.0 (Compatibility Mode)

## Update: Pioneer DDJ REV1 mapping v2.1

## To Use it:

Copy the `Pioneer-DDJ-REV1-script.js` and `Pioneer DDJ-REV1.midi.xml` files into your User Mapping folder. 

### Windows
    %LOCALAPPDATA%\Mixxx
### macOS
    Mixxx 2.3: ~/Library/Containers/org.mixxx.mixxx/Data/Library/Application Support/Mixxx
    Mixxx 2.2 and earlier: ~/Library/Application Support/Mixxx
### Linux
    ~/.mixxx/
    This resolves to /home/<your-username>/.mixxx
    


**For more infomration follow the** [official Mixxx guide](https://manual.mixxx.org/2.6/en/chapters/controlling_mixxx.html#installing-a-mapping-from-the-forum).
